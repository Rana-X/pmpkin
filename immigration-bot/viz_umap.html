About This Visualization

What is UMAP?

  • Uniform Manifold Approximation and Projection
  • Reduces 3072-dimensional embeddings to 2D
  • Similar cases cluster together
  • Preserves both local and global structure

Data Pipeline

  • 1. Extract text from PDFs (Reducto)
  • 2. Extract features (GPT-4o)
  • 3. Generate embeddings (OpenAI)
  • 4. Reduce to 2D (UMAP)

Interpretation

  • Points close together = similar cases
  • Clusters = common patterns
  • Hover for case details
  • Click and drag to zoom
